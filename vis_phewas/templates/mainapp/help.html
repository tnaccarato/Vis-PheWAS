Help

Controls

Use the mouse to pan and the scroll-wheel to zoom the plot.

Click on a node to unfold it to the next level. If a node is unfolded, click it again to refold it.

Right-click a node to force the label to be displayed.

Allele Details

Click on an allele node to display a table with detailed information about the node.

Filters

Use the add filter button to add a filter. You can filter by gene, chromosome, or p-value. You can also filter by the number of SNPs in a gene. You can add multiple filters and they will be combined with an AND operator.
