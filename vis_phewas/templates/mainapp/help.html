Help

Controls

Use the mouse to pan and the scroll-wheel to zoom the plot.

Click on a node to unfold it to the next level. If a node is unfolded, click it again to refold it.

Right-click a node to force the label to be displayed.

Click on a node to display a table with detailed information about the node.

Allele Details

Click on an allele node to display a table with detailed information about the node.

From the table you can click to display other diseases that are associated with the allele as well as view the highest associated odds ratio and p-value for that allele. By clicking on a disease in the table, you can add the disease to the filter list.

Filters

Use the add filter button to add a filter. You can filter by gene, chromosome, or p-value. You can also filter by the number of SNPs in a gene. You can add multiple filters which will be combined with an AND operator.
